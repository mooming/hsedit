# PLAND.md — MakeBuilder Integration

## Goal
Configure hsedit to use makebuilder for automatic CMake generation. makebuilder scans directory structure + `.module.config` files and generates `CMakeLists.txt` automatically.

## Steps

### 0. makebuilder as submodule (DONE)
- `external/makebuilder` is now a git submodule

### 1. Build makebuilder
- Run `build.sh -release` inside `external/makebuilder/`
- Produces the `makebuild` binary in `Application/MakeBuild/Release/`

### 2. Restructure project layout
- Move `src/main.cpp` → `app/main.cpp`
- `src/` becomes a pure StaticLibrary module (module_registry, event_bus)
- `app/` becomes the hsedit Executable module

### 3. Create configuration files
- `.project.config` — global settings (C++17, compile flags, ncurses via customCMake)
- `.module.config` (root) — `buildType = None`
- `src/.module.config` — `buildType = StaticLibrary`, name = libhsedit
- `app/.module.config` — `buildType = Executable`, name = hsedit, dependency = libhsedit
- `modules/terminal/.module.config` — `buildType = Executable`, library = ncurses
- `tests/.module.config` — `buildType = Executable`, name = hsedit_tests, dependency = libhsedit, library = ncurses

### 4. Run makebuilder
- `./external/makebuilder/run.sh /path/to/hsedit`
- Generates `CMakeLists.txt` at project root

### 5. Build and verify
- `cmake -B build -S . -G "Ninja Multi-Config"`
- `cmake --build build --config Release`
- Verify all three targets build: `hsedit`, `terminal_module`, `hsedit_tests`

## Scope
- **IN**: makebuilder integration, config files, project restructuring (src/app split)
- **OUT**: fixing/rewriting source code, adding features, changing test logic

## Verification
- [ ] makebuilder builds successfully
- [ ] CMakeLists.txt is generated by makebuilder
- [ ] `hsedit` executable builds (links src library + main.cpp)
- [ ] `terminal_module` executable builds (with ncurses)
- [ ] `hsedit_tests` executable builds (links src library + ncurses)
- [ ] `cmake --build build --config Dev` works
- [ ] `cmake --build build --config Debug` works
