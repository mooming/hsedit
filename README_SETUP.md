# C++ Development Setup with Vim

## Installed Tools

### Core Tools
- **ctags** (v5.8_2) - Tag-based navigation
- **LLVM** (v22.1.8) - Includes clangd and lldb
- **Vim** (v9.2.0750) - Text editor

### Vim Plugins
- **coc.nvim** - Language Server Protocol support
- **syntastic** - Linting
- **vim-fugitive** - Git integration
- **vim-surround** - Surround text with brackets/quotes
- **vim-commentary** - Easy commenting
- **vim-speeddating** - Better date/time editing
- **vim-repeat** - Make plugins work with `.`

### Language Server
- **clangd** (via LLVM) - C++ language server

### Debug Tools
- **lldb** (via LLVM) - Debugger
- **gdb** - GNU Debugger (system installed)

## Configuration Files

### ~/.vimrc
Main Vim configuration with:
- C++ syntax highlighting
- LSP integration (clangd)
- Build/run shortcuts
- Debug keybindings
- Navigation helpers

### ~/.vim/coc-settings.json
LLVM configuration for clangd language server.

## Quick Start

### 1. Install Tags
```bash
cd your/project
ctags -R .
```

### 2. Copy Configuration
```bash
cp .vimrc ~/.vimrc
```

### 3. Install coc.nvim Extension
Open Vim and run:
```vim
:CocInstall coc-clangd
```

## Build Examples

### Simple Build
```bash
clang++ -std=c++17 -Wall -Wextra -g main.cpp -o main
./main
```

### CMake
```bash
mkdir build && cd build
cmake ..
make
./your_binary
```

### Makefile
```makefile
CXX = clang++
CXXFLAGS = -std=c++17 -Wall -Wextra -g
TARGET = main

all: $(TARGET)

$(TARGET): main.cpp
	$(CXX) $(CXXFLAGS) $< -o $@

clean:
	rm -f $(TARGET)

run: $(TARGET)
	./$(TARGET)
```

## Troubleshooting

### clangd not found
```bash
/opt/homebrew/Cellar/llvm/22.1.8/bin/clangd --version
```

### Tags not working
```bash
ctags -R .
```

### coc.nvim not loading
```vim
:CocInfo
:CocList extensions
```
