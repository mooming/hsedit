Overview

This guide provides a complete setup for C++ development using Vim editor with modern tooling including Language Server Protocol (LSP), debugging, and build integration.

📦 Core Tools

  • Vim 9.2 - Text editor
  • ctags 5.8 - Tag navigation
  • LLVM 22.1 - Compiler & tools

🔧 Vim Plugins

  • coc.nvim - LSP support
  • syntastic - Linting
  • vim-fugitive - Git integration

🎯 Key Features

  • IntelliSense via clangd
  • Build & run shortcuts
  • Debug with lldb/gdb

Installation

Step 1: Install Core Tools Installed

$ brew install ctags
$ brew install llvm
$ brew install vim

Step 2: Install Vim Plugins Installed

$ mkdir -p ~/.vim/bundle
$ cd ~/.vim/bundle

$ git clone https://github.com/neoclide/coc.nvim.git --depth 1
$ git clone https://github.com/tpope/vim-fugitive.git --depth 1
$ git clone https://github.com/vim-syntastic/syntastic.git --depth 1
$ git clone https://github.com/tpope/vim-surround.git --depth 1
$ git clone https://github.com/tpope/vim-commentary.git --depth 1

Step 3: Install clangd Extension Required

vim +CocInstall coc-clangd +qa
ℹ️ Note: clangd is included with LLVM installation. The extension connects Vim to clangd for intelligent code analysis.

Configuration

Files Created

~/.vimrc

Main Vim configuration with C++ settings, keybindings, and plugin configuration.

⚠️ Must be in home directory (~), not project directory!

~/.vim/coc-settings.json

clangd language server configuration with file type mappings and root patterns.

Configuration Details

// .vimrc - Key sections
filetype plugin indent on
syntax enable

" Leader key
let mapleader = ","

" LSP configuration
let g:coc_config_home = $HOME . '/.vim/coc-settings.json'
let g:coc_global_extensions = ['coc-clangd']

" Build shortcuts
nnoremap bb :w !clang++ -std=c++17 -Wall -Wextra -g % 
nnoremap rr :!./tmp/%<

LLVM Configuration

{
  "languageserver": {
    "clangd": {
      "command": "/opt/homebrew/Cellar/llvm/22.1.8/bin/clangd",
      "filetypes": ["c", "cpp"],
      "rootPatterns": [
        "compile_commands.json", 
        ".git/", 
        "CMakeLists.txt",
        "Makefile"
      ],
      "initializationOptions": {
        "clangdFileStatus": true
      }
    }
  }
}

Keybindings Reference

Navigation Active

KeyAction
,gdGo to definition
,gyGo to type definition
,giGo to implementation
,grShow references
,KShow documentation

Build & Run Active

KeyAction
,bbBuild with project flags (.build_flags)
,bdBuild without extra flags
,rrRecompile and run (with project flags)
,rxRun without recompiling
,bmBuild all (make)
ℹ️ Project Build Flags: Create a .build_flags file in your project directory to specify compiler flags (e.g., -lncurses, -lpthread, etc.). The ,bb and ,rr commands automatically read this file.
