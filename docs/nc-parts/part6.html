Quick Reference: Essential Functions

FunctionWhat it does
initscr()Initialize ncurses. Call first.
endwin()Restore terminal. Call last.
cbreak()Each key press arrives immediately.
noecho()Don't echo typed characters.
keypad(stdscr, TRUE)Enable special key codes.
getch()Get a key press (blocking).
nodelay(win, TRUE)Non-blocking getch (returns ERR).
timeout(ms)Blocking getch with timeout.
printw("fmt", ...)Print formatted text at cursor.
mvprintw(y, x, "fmt", ...)Print at specific position.
wprintw(win, "fmt", ...)Print to a window.
refresh()Draw everything to terminal.
wrefresh(win)Draw a specific window.
erase()Clear the screen.
move(y, x)Move cursor.
curs_set(1)Show cursor (0=hidden).
flushinp()Clear input buffer.
start_color()Enable color support.
init_pair(id, fg, bg)Define a color pair.
attron(COLOR_PAIR(id))Apply a color pair.
KEY_RESIZETerminal was resized.

Common Mistakes

  • Forgetting refresh(): ncurses buffers output. Without refresh, nothing appears on screen.
  • Forgetting keypad(stdscr, TRUE): Arrow keys and F-keys won't work.
  • Coordinates are (y, x): Row first, then column. Not (x, y).
  • Not calling endwin() on crash: Terminal is left broken. Use a destructor or atexit handler.
  • Using printw after a keypress: printw changes the cursor position. If you're drawing in a specific spot, you may need to reposition.
