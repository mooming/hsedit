Initialization & Termination

Home→ Initialization

initscr()

Initializes ncurses mode. Must be called before any other ncurses function.

initscr(); // Now you can use ncurses functions printw("Hello, ncurses!"); refresh(); getch(); endwin();

endwin()

Restores the terminal to its previous state. Should be called before program exit.

// ... use ncurses ... refresh(); getch(); endwin(); // Restore terminal

refresh()

Updates the physical screen to match the virtual screen (stdscr).

mvprintw(10, 20, "Updated at %ld", time(NULL)); refresh(); // Forces screen update

doupdate()

Updates the screen only after explicit calls to wnoutrefresh().

mvprintw(0, 0, "Screen 1"); wnoutrefresh(stdscr); mvprintw(10, 10, "Screen 2"); wnoutrefresh(stdscr); doupdate(); // Both updates applied here

newterm(const char *type, FILE *outfd, FILE *infd)

Alternative to initscr() for multiple screen types.

TERMINAL *term = newterm((char *)"xterm", stdout, stdin); initscr(); // ... use ncurses ... endwin(); delterm();

delterm(TERMINAL *term)

Deletes the terminal associated with the current screen.
