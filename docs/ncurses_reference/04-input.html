Input Handling

Home→ Input

getch()

Read a single character from stdscr (blocking).

int ch = getch(); if (ch == KEY_UP) { // Handle up arrow }

wgetch(WINDOW *win)

Read a character from a window.

WINDOW *win = newwin(10, 40, 0, 0); int ch = wgetch(win);

mvgetch(int y, int x)

Move cursor and read a character from stdscr.

mvwgetch(WINDOW *win, int y, int x)

Move cursor in window and read a character.

nodelay(WINDOW *win, bool delay)

Set non-blocking mode for input.

nodelay(stdscr, TRUE); int ch = getch(); if (ch == ERR) { // No input available }

halfdelay(int tenths)

Set half-delay mode (timeout in tenths of seconds).

cbreak()

Disables line buffering (characters available immediately).

nocbreak()

Re-enables line buffering.

echo()

Enables echo (characters typed are displayed).

noecho()

Disables echo.

raw()

Disables all input processing (even more raw than cbreak).

ungetch(int ch)

Push a character back into the input buffer.

ungetch('q'); // Next getch() returns 'q'

keypad(WINDOW *win, bool bf)

Enable/disable keypad mode for special keys.

keypad(stdscr, TRUE); int ch = getch(); if (ch == KEY_UP) { ... }

set_escdelay(int ms)

Set the timeout for escape sequences.

ESCDELAY

Global variable for escape sequence timeout.

getmouse(MOUSEEVENT *event)

Get mouse event (requires mousemask() first).

ungetmouse(MOUSEEVENT *event)

Push a mouse event back into the buffer.

mousemask(mmask_t newmask, mmask_t *oldmask)

Enable/disable mouse events.

mousemask(BUTTON1_CLICKED | BUTTON1_DOUBLE_CLICKED, NULL); getch(); // Will now receive mouse events

wgetmouse(WINDOW *win, int *y, int *x)

Get the last mouse position.

is_pad(WINDOW *win)

Returns TRUE if win is a pad.

is_termresized(WINDOW *win, int y, int x)

Returns TRUE if the terminal has been resized.
