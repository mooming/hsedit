Color Support

Home› Color

start_color()

Initializes color support. Must be called before using colors.

start_color();
init_pair(1, COLOR_RED, COLOR_BLACK);
attron(COLOR_PAIR(1));
printw("Red text on black");
attroff(COLOR_PAIR(1));
refresh();

has_colors()

Returns TRUE if the terminal supports color.

init_pair(short pair, short f, short b)

Initializes a color pair.

init_color(short color, short r, short g, short b)

Initializes a color with RGB values.

init_pair_nocolor(short pair, chtype attr)

Initializes a color pair without colors.

color_content(short color, short* r, short* g, short* b)

Returns the RGB content of a color.

color_pair(short pair)

Returns the attribute for a color pair.

pair_content(short pair, short* f, short* b)

Returns the foreground/background colors of a pair.

attron(chtype attr)

TURNS on an attribute on stdscr.

attron(A_BOLD);
printw("Bold");
attroff(A_BOLD);

attroff(chtype attr)

TURNS off an attribute on stdscr.

attrset(chtype attr)

Sets the attribute on stdscr.

attr_get(chtype* attr, short* pair, void* opts)

Returns the current attribute.

attr_off(chtype attr, void* opts)

TURNS off attributes on stdscr.

attr_on(chtype attr, void* opts)

TURNS on attributes on stdscr.

color_set(short pair, void* opts)

Sets the color for the next text output.
