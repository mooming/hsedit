Terminal & Screen Management

Home→ Terminal

cols

Global variable: number of columns in the terminal.

printw("Terminal is %d columns wide", COLS);

lines

Global variable: number of lines in the terminal.

printw("Terminal is %d lines tall", LINES);

CR

Character constant for carriage return.

LF

Character constant for line feed.

TAB

Character constant for tab.

BEL

Character constant for bell.

BACKSPACE

Character constant for backspace.

ESC

Character constant for escape.

SIGWINCH

Signal sent when terminal is resized.

resize_term(int nlines, int ncols)

Resize the terminal to new dimensions.

is_term_resized(int nlines, int ncols)

Returns TRUE if the terminal was resized.

set_term(TERMINAL *term)

Sets the current terminal.

newterm(const char *type, FILE *outfd, FILE *infd)

Creates a new terminal.

delterm(TERMINAL *term)

Deletes a terminal.

termaddr(int addr)

Returns the address of a terminal.

termattrs()

Returns the attributes supported by the terminal.

termname()

Returns the terminal name.

newpad(int nlines, int ncols)

Creates a pad (scrolling window).

WINDOW *pad = newpad(100, 80); prefresh(pad, 0, 0, 0, 0, 20, 79);

prefresh(WINDOW *pad, int pminrow, int pmincol, int sminrow, int smincol, int smaxrow, int smaxcol)

Refreshes a portion of a pad to the screen.

pnoutrefresh(WINDOW *pad, int pminrow, int pmincol, int sminrow, int smincol, int smaxrow, int smaxcol)

Like prefresh() but doesn't call doupdate().
