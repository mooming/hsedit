Utility Functions

Home→ Utility

getyx(WINDOW *win, int *y, int *x)

Returns the current cursor position in a window.

WINDOW *win = initscr(); int y, x; getyx(win, y, x); printw("Cursor at (%d, %d)", y, x);

getbegyx(WINDOW *win, int *y, int *x)

Returns the top-left corner position of a window.

getmaxyx(WINDOW *win, int *y, int *x)

Returns the maximum dimensions of a window.

getparyx(WINDOW *win, int *y, int *x)

Returns the position of a window relative to its parent.

mouse_trafo(int *y, int *x, int to_screen)

Converts between screen and window coordinates.

use_env(bool flag)

Enables/disables use of $TERM and $COLUMNS/$LINES.

use_default_colors()

Sets default colors to -1 (transparent).

setscrreg(int top, int bottom)

Sets the scroll region for stdscr.

setscrreg(5, 20); // Lines 5-20 are scrollable

wsetscrreg(WINDOW *win, int top, int bottom)

Sets the scroll region for a window.

scrollok(WINDOW *win, bool bf)

Enables/disables scrolling in a window.

wborder_set(WINDOW *win, ...)

Draws a border using Unicode characters.

set_tabsize(int size)

Sets the tab size (default is 8).

typeahead(int fildes)

Sets the file descriptor for type-ahead checking.

meta(WINDOW *win, bool bf)

Enables/disables 8-bit input mode.

napms(int ms)

Pauses execution for ms milliseconds.

delay_output(int ms)

Pauses for ms milliseconds (output-based).

clear()

Clears stdscr and marks it as needing refresh.

clrtobot()

Clears from cursor to bottom of screen.

clrtoeol()

Clears from cursor to end of line.
