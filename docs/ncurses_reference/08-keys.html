Key Definitions

Home→ Keys

Arrow Keys

KEY_UP

Up arrow key.

if (ch == KEY_UP) { ... }

KEY_DOWN

Down arrow key.

KEY_LEFT

Left arrow key.

KEY_RIGHT

Right arrow key.

Function Keys

KEY_F(0) through KEY_F(15)

Function keys F1 through F16.

if (ch == KEY_F(1)) { // F1 pressed }

KEY_BTAB

BackTab key (Shift+Tab).

KEY_B1

Button 1 (mouse).

Navigation Keys

KEY_HOME

Home key.

KEY_END

End key.

KEY_PPAGE

Page Up key.

KEY_NPAGE

Page Down key.

Editing Keys

KEY_ENTER

Enter key.

KEY_BACKSPACE

Backspace key.

KEY_IC

Insert key.

KEY_DC

Delete key.

KEY_NPAGE

Next page (Page Down).

Mouse Constants

BUTTON1_PRESSED

Left mouse button pressed.

BUTTON1_RELEASED

Left mouse button released.

BUTTON1_CLICKED

Left mouse button clicked.

BUTTON1_DOUBLE_CLICKED

Left mouse button double-clicked.

BUTTON1_TRIPLE_CLICKED

Left mouse button triple-clicked.

BUTTON1_CLICKED

Left mouse button clicked.

Special Constants

ERR

Error indicator (-1).

if (ch == ERR) { // Error occurred }

KEY_RESIZE

Terminal was resized.

KEY_MOUSE

Mouse event occurred.
