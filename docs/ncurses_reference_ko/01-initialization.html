초기화 및 종료

홈› 초기화

initscr()

ncurses 모드를 초기화합니다. 다른 모든 ncurses 함수 호출 전에 반드시 호출해야 합니다.

initscr();
// 이제 ncurses 함수를 사용할 수 있습니다
printw("Hello, ncurses!");
refresh();
getch();
endwin();

endwin()

터미널을 이전 상태로 복원합니다. 프로그램 종료 전에 호출해야 합니다.

// ... ncurses 사용 ...
refresh();
getch();
endwin();

refresh()

물리적 화면을 가상 화면 (stdscr) 과 일치시키도록 업데이트합니다.

mvprintw(10, 20, "Updated at %ld", time(NULL));
refresh();

doupdate()

wnoutrefresh() 호출 후에만 화면을 업데이트합니다.

mvprintw(0, 0, "Screen 1");
wnoutrefresh(stdscr);

mvprintw(10, 10, "Screen 2");
wnoutrefresh(stdscr);

doupdate();

newterm(const char* type, FILE* outfd, FILE* infd)

다중 화면 유형에 대한 initscr() 대체 함수입니다.

TERMINAL* term = newterm(nullptr, stdout, stdin);
initscr();
// ... ncurses 사용 ...
endwin();
delterm();

delterm(TERMINAL* term)

현재 화면에 연결된 터미널을 삭제합니다.
