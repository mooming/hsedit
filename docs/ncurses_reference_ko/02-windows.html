윈도우 관리

홈› 윈도우

newwin(int nlines, int ncols, int begin_y, int begin_x)

새로운 윈도우를 생성합니다.

WINDOW* subwin = newwin(5, 40, 5, 10); if (subwin) { wprintw(subwin, "서브윈도우 내용"); wrefresh(subwin); }

delwin(WINDOW* win)

윈도우를 삭제하고 메모리를 해제합니다.

WINDOW* win = newwin(10, 20, 0, 0); // ... 윈도우 사용 ... delwin(win);

mvwin(WINDOW* win, int y, int x)

윈도우를 새 위치로 이동합니다.

WINDOW* win = newwin(5, 20, 0, 0); mvprintw(0, 0, "이동 전"); mvwin(win, 10, 10); mvprintw(0, 0, "이동 후"); refresh();

wresize(WINDOW* win, int nlines, int ncols)

기존 윈도우의 크기를 변경합니다.

WINDOW* win = newwin(5, 20, 0, 0); wresize(win, 10, 40); mvprintw(0, 0, "크기 변경된 윈도우"); wrefresh(win);

subwin(WINDOW* orig, int nlines, int ncols, int begin_y, int begin_x)

원본 윈도우와 메모리를 공유하는 서브윈도우를 생성합니다.

WINDOW* main_win = initscr(); WINDOW* sub = subwin(main_win, 5, 20, 5, 5); wattron(sub, A_REVERSE); wprintw(sub, "서브윈도우"); wrefresh(sub);

derwin(WINDOW* orig, int nlines, int ncols, int begin_y, int begin_x)

subwin() 과 유사하지만 메모리를 공유하지 않습니다.

mvderwin(WINDOW* win, int pary, int parx)

다른 윈도우에서 파생된 윈도우를 이동합니다.

dupwin(WINDOW* win)

윈도우의 정확한 복제본을 생성합니다.

window_ok(WINDOW* win)

win 이 유효한 윈도우 포인터이면 true 를 반환합니다.

touchwin(WINDOW* win)

윈도우의 모든 줄을 새로고침이 필요하도록 표시합니다.

WINDOW* win = newwin(10, 40, 0, 0); touchwin(win); wrefresh(win);

touchline(WINDOW* win, int start, int count)

윈도우의 줄 범위를 새로고침이 필요하도록 표시합니다.

untouchwin(WINDOW* win)

윈도우의 모든 줄을 새로고침이 필요하지 않도록 표시합니다.

redrawwin(WINDOW* win)

윈도우 전체의 새로고침을 강제합니다.

redrawline(WINDOW* win, int start, int count)

줄 범위의 새로고침을 강제합니다.

wclear(WINDOW* win)

윈도우를 지웁니다 (공백으로 채움).

werase(WINDOW* win)

wclear() 와 유사하지만 윈도우를 새로고침 필요로 표시합니다.

waddch(WINDOW* win, chtype ch)

윈도우에 문자를 추가합니다.

WINDOW* win = newwin(5, 20, 0, 0); waddch(win, 'A'); waddch(win, 'B'); wrefresh(win);

wmove(WINDOW* win, int y, int x)

윈도우 내에서 커서를 이동합니다.

WINDOW* win = newwin(10, 40, 0, 0); wmove(win, 5, 10); wprintw(win, "커서 위치 5,10"); wrefresh(win);

wrefresh(WINDOW* win)

물리적 화면을 윈도우와 일치시키도록 업데이트합니다.

WINDOW* win = newwin(10, 40, 0, 0); wprintw(win, "내용"); wrefresh(win);

wnoutrefresh(WINDOW* win)

wrefresh() 와 유사하지만 doupdate() 를 호출하지 않습니다.

WINDOW* win1 = newwin(5, 20, 0, 0); WINDOW* win2 = newwin(5, 20, 0, 25); wprintw(win1, "윈도우 1"); wprintw(win2, "윈도우 2"); wnoutrefresh(win1); wnoutrefresh(win2); doupdate();

wborder(WINDOW* win, ...)

윈도우 주변에 테두리를 그립니다.

WINDOW* win = newwin(10, 40, 5, 5); wborder(win, '|', '|', '-', '-', '+', '+', '+', '+'); wrefresh(win);

wattron(WINDOW* win, attr_t attr)

속성 (굵게, 역전 등) 을 켭니다.

WINDOW* win = newwin(5, 20, 0, 0); wattron(win, A_BOLD); wprintw(win, "굵은 텍스트"); wattroff(win, A_BOLD); wrefresh(win);

wattroff(WINDOW* win, attr_t attr)

속성을 끕니다.
