문자 및 텍스트 I/O

홈› 텍스트 I/O

printw(const char* fmt, ...)

stdscr 에 서식 있는 출력을 수행합니다 (printf 와 유사).

int x = 42; printw("정답은 %d 입니다", x); refresh();

wprintw(WINDOW* win, const char* fmt, ...)

윈도우에 서식 있는 출력을 수행합니다.

WINDOW* win = newwin(5, 20, 0, 0); wprintw(win, "값: %d", 42); wrefresh(win);

mvprintw(int y, int x, const char* fmt, ...)

위치로 이동하여 stdscr 에 출력을 수행합니다.

mvprintw(10, 20, "10,20 에 있는 텍스트"); refresh();

mvwprintw(WINDOW* win, int y, int x, const char* fmt, ...)

윈도우에서 위치로 이동하여 출력을 수행합니다.

addch(chtype ch)

stdscr 에 문자를 추가합니다.

addch('A'); addch('B' | A_BOLD); // 굵은 B refresh();

waddch(WINDOW* win, chtype ch)

윈도우에 문자를 추가합니다.

mvaddch(int y, int x, chtype ch)

위치로 이동하여 stdscr 에 문자를 추가합니다.

mvwaddch(WINDOW* win, int y, int x, chtype ch)

윈도우에서 위치로 이동하여 문자를 추가합니다.

addstr(const char* str)

stdscr 에 문자열을 추가합니다.

addstr("안녕하세요!"); refresh();

waddstr(WINDOW* win, const char* str)

윈도우에 문자열을 추가합니다.

mvaddstr(int y, int x, const char* str)

위치로 이동하여 stdscr 에 문자열을 추가합니다.

mvwaddstr(WINDOW* win, int y, int x, const char* str)

윈도우에서 위치로 이동하여 문자열을 추가합니다.

slk_init(int format)

소프트 키를 초기화합니다 (3270 터미널).

slk_init(SLK_STDMODE); slk_set(1, 1, "도움말", 1); slk_set(2, 1, "나가기", 1); slk_refresh();

slk_set(int labnum, int align, const char* label, int refresh)

소프트 키 레이블을 설정합니다.

slk_refresh()

소프트 키를 다시 그립니다.

slk_clear()

모든 소프트 키를 지웁니다.

slk_restore()

소프트 키를 복원합니다.
