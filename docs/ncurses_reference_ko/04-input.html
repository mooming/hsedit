입력 처리

홈› 입력

getch()

stdscr 에서 단일 문자를 읽습니다 (블로킹).

int ch = getch();
if (ch == KEY_UP)
{
    // 방향 키 위 처리
}

wgetch(WINDOW* win)

윈도우에서 문자를 읽습니다.

WINDOW* win = newwin(10, 40, 0, 0);
int ch = wgetch(win);

mvgetch(int y, int x)

커서를 이동하고 stdscr 에서 문자를 읽습니다.

mvwgetch(WINDOW* win, int y, int x)

윈도우에서 커서를 이동하고 문자를 읽습니다.

nodelay(WINDOW* win, bool delay)

입력에 대한 비블로킹 모드를 설정합니다.

nodelay(stdscr, TRUE);
int ch = getch();
if (ch == ERR)
{
    // 입력 불가
}

halfdelay(int tenths)

하프딜레이 모드를 설정합니다 (1/10 초 단위).

cbreak()

줄 버퍼링을 비활성화합니다 (문자가 즉시 사용 가능).

nocbreak()

줄 버퍼링을 다시 활성화합니다.

echo()

에코를 활성화합니다 (입력한 문자가 표시됩니다).

noecho()

에코를 비활성화합니다.

raw()

모든 입력 처리를 비활성화합니다 (cbreak 보다 더 원시적).

ungetch(int ch)

문자를 입력 버퍼에 다시 밀어넣습니다.

ungetch('q');  // 다음 getch() 가 'q' 반환

keypad(WINDOW* win, bool bf)

특수 키에 대한 키패드 모드를 활성화/비활성화합니다.

keypad(stdscr, TRUE);
int ch = getch();
if (ch == KEY_UP) { ... }

set_escdelay(int ms)

이스케이프 시퀀스의 타임아웃을 설정합니다.

ESCDELAY

이스케이프 시퀀스 타임아웃에 대한 전역 변수.

getmouse(MOUSEEVENT* event)

마우스 이벤트를 가져옵니다 (먼저 mousemask() 필요).

ungetmouse(MOUSEEVENT* event)

마우스 이벤트를 버퍼에 다시 밀어넣습니다.

mousemask(mmask_t newmask, mmask_t* oldmask)

마우스 이벤트를 활성화/비활성화합니다.

mousemask(BUTTON1_CLICKED | BUTTON1_DOUBLE_CLICKED, nullptr);
getch();  // 이제 마우스 이벤트 수신

wgetmouse(WINDOW* win, int* y, int* x)

마우스의 마지막 위치를 가져옵니다.

is_pad(WINDOW* win)

win 이 패드이면 true 를 반환합니다.

is_termresized(WINDOW* win, int y, int x)

터미널이 크기 조정되었으면 true 를 반환합니다.
