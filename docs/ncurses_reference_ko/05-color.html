색상 지원

홈› 색상

start_color()

색상 지원을 초기화합니다. 색상 사용 전에 반드시 호출해야 합니다.

start_color();
init_pair(1, COLOR_RED, COLOR_BLACK);
attron(COLOR_PAIR(1));
printw("검정 배경에 빨간 텍스트");
attroff(COLOR_PAIR(1));
refresh();

has_colors()

터미널이 색상을 지원하면 true 를 반환합니다.

init_pair(short pair, short f, short b)

색상 쌍을 초기화합니다.

init_color(short color, short r, short g, short b)

RGB 값으로 색상을 초기화합니다.

init_pair_nocolor(short pair, chtype attr)

색상 없이 색상 쌍을 초기화합니다.

color_content(short color, short* r, short* g, short* b)

색상의 RGB 콘텐츠를 반환합니다.

color_pair(short pair)

색상 쌍에 대한 속성을 반환합니다.

pair_content(short pair, short* f, short* b)

색상 쌍의 전경/배경 색상을 반환합니다.

attron(chtype attr)

stdscr 에 속성을 켭니다.

attron(A_BOLD);
printw("굵게");
attroff(A_BOLD);

attroff(chtype attr)

stdscr 에서 속성을 끕니다.

attrset(chtype attr)

stdscr 에 속성을 설정합니다.

attr_get(chtype* attr, short* pair, void* opts)

현재 속성을 반환합니다.

attr_off(chtype attr, void* opts)

stdscr 에서 속성을 끕니다.

attr_on(chtype attr, void* opts)

stdscr 에 속성을 켭니다.

color_set(short pair, void* opts)

다음 텍스트 출력에 대한 색상을 설정합니다.
