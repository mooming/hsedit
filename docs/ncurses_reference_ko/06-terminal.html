터미널 및 화면 관리

홈› 터미널

COLS

전역 변수: 터미널의 열 수.

printw("터미널은 %d 열 너비입니다", COLS);

LINES

전역 변수: 터미널의 줄 수.

printw("터미널은 %d 줄 높이입니다", LINES);

CR

캐리지 리턴 문자 상수.

LF

라인 피드 문자 상수.

TAB

탭 문자 상수.

BEL

벨 문자 상수.

BACKSPACE

백스페이스 문자 상수.

ESC

이스케이프 문자 상수.

SIGWINCH

터미널이 크기 조정될 때 보내지는 시그널.

resize_term(int nlines, int ncols)

터미널을 새 크기로 크기 조정합니다.

is_term_resized(int nlines, int ncols)

터미널이 크기 조정되었으면 true 를 반환합니다.

set_term(TERMINAL* term)

현재 터미널을 설정합니다.

newterm(const char* type, FILE* outfd, FILE* infd)

새 터미널을 생성합니다.

delterm(TERMINAL* term)

터미널을 삭제합니다.

termaddr(int addr)

터미널의 주소를 반환합니다.

termattrs()

터미널에서 지원하는 속성을 반환합니다.

termname()

터미널 이름을 반환합니다.

newpad(int nlines, int ncols)

패드 (스크롤 윈도우) 를 생성합니다.

WINDOW* pad = newpad(100, 80);
prefresh(pad, 0, 0, 0, 0, 20, 79);

prefresh(WINDOW* pad, int pminrow, int pmincol, int sminrow, int smincol, int smaxrow, int smaxcol)

패드의 일부를 화면에 새로고침합니다.

pnoutrefresh(WINDOW* pad, int pminrow, int pmincol, int sminrow, int smincol, int smaxrow, int smaxcol)

prefresh() 와 유사하지만 doupdate() 를 호출하지 않습니다.
