유틸리티 함수

홈› 유틸리티

getyx(WINDOW* win, int* y, int* x)

윈도우의 현재 커서 위치를 반환합니다.

WINDOW* win = initscr(); int y, x; getyx(win, y, x); printw("커서 위치 (%d, %d)", y, x);

getbegyx(WINDOW* win, int* y, int* x)

윈도우의 위쪽 왼쪽 모서리 위치를 반환합니다.

getmaxyx(WINDOW* win, int* y, int* x)

윈도우의 최대 크기를 반환합니다.

getparyx(WINDOW* win, int* y, int* x)

부모에 대한 상대적 윈도우 위치를 반환합니다.

mouse_trafo(int* y, int* x, int to_screen)

화면과 윈도우 좌표 간 변환을 수행합니다.

use_env(bool flag)

$TERM 및 $COLUMNS/$LINES 사용 여부를 활성화/비활성화합니다.

use_default_colors()

기본 색상을 -1 (투명) 로 설정합니다.

setscrreg(int top, int bottom)

stdscr 의 스크롤 영역을 설정합니다.

setscrreg(5, 20); // 줄 5-20 이 스크롤 가능

wsetscrreg(WINDOW* win, int top, int bottom)

윈도우의 스크롤 영역을 설정합니다.

scrollok(WINDOW* win, bool bf)

윈도우에서 스크롤링을 활성화/비활성화합니다.

wborder_set(WINDOW* win, ...)

유니코드 문자를 사용하여 테두리를 그립니다.

set_tabsize(int size)

탭 크기를 설정합니다 (기본값은 8).

typeahead(int fildes)

타입 어헤드 검사를 위한 파일 설명자를 설정합니다.

meta(WINDOW* win, bool bf)

8 비트 입력 모드를 활성화/비활성화합니다.

napms(int ms)

ms 밀리초 동안 실행을 중단합니다.

delay_output(int ms)

출력에 기반하여 ms 밀리초 동안 중단합니다.

clear()

stdscr 을 지우고 새로고침 필요로 표시합니다.

clrtobot()

커서부터 화면 하단까지 지웁니다.

clrtoeol()

커서부터 줄 끝까지 지웁니다.
