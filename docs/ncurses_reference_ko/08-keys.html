키 정의

홈› 키

방향 키

KEY_UP

위쪽 화살표 키.

if (ch == KEY_UP) { ... }

KEY_DOWN

아래쪽 화살표 키.

KEY_LEFT

왼쪽 화살표 키.

KEY_RIGHT

오른쪽 화살표 키.

함수 키

KEY_F(0) ~ KEY_F(15)

함수 키 F1 ~ F16.

if (ch == KEY_F(1)) { // F1 누름 }

KEY_BTAB

BackTab 키 (Shift+Tab).

KEY_B1

마우스 버튼 1.

네비게이션 키

KEY_HOME

Home 키.

KEY_END

End 키.

KEY_PPAGE

Page Up 키.

KEY_NPAGE

Page Down 키.

편집 키

KEY_ENTER

Enter 키.

KEY_BACKSPACE

Backspace 키.

KEY_IC

Insert 키.

KEY_DC

Delete 키.

마우스 상수

BUTTON1_PRESSED

왼쪽 마우스 버튼 눌림.

BUTTON1_RELEASED

왼쪽 마우스 버튼 놓임.

BUTTON1_CLICKED

왼쪽 마우스 버튼 클릭.

BUTTON1_DOUBLE_CLICKED

왼쪽 마우스 버튼 더블클릭.

BUTTON1_TRIPLE_CLICKED

왼쪽 마우스 버튼 트리플클릭.

특수 상수

ERR

오류 표시자 (-1).

if (ch == ERR) { // 오류 발생 }

KEY_RESIZE

터미널이 크기 조정됨.

KEY_MOUSE

마우스 이벤트 발생.
